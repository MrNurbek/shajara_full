# Texnik audit va rivojlantirish xaritasi

Ushbu arxivda loyiha quyidagi yo‘nalishlarda mukammallashtirildi:

1. Daraxt qurish algoritmidagi `seen_marriages` state xatosi tuzatildi.
2. `/api/tree/?max_depth=...` endpointida noto‘g‘ri qiymat 500 xato bermaydigan qilindi.
3. UI to‘liq yangilandi: sidebar, topbar, statistik kartalar, modern ranglar, modal formalar.
4. Staff foydalanuvchi uchun frontenddan shaxs qo‘shish/tahrirlash endpointlari qo‘shildi.
5. Nikoh qo‘shish va farzand bog‘lash API endpointlari qo‘shildi.
6. Dockerfile xatoni yashirmaydigan holatga keltirildi.
7. Testlar qo‘shildi.

Keyingi tavsiya: katta shajaralar uchun nested HTML o‘rniga SVG/D3 yoki React Flow asosidagi lazy-rendering mexanizmiga o‘tish.
